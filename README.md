# Machine Learning Short Notes

This repository contains a beginner-friendly summary of the core concepts from the **Machine Learning Specialization by Andrew Ng**.

These notes were created while learning the course and are intended as a quick revision guide for students and beginners.

## Topics Covered

- Linear Regression
- Logistic Regression
- Gradient Descent
- Neural Networks
- Decision Trees
- Clustering (K-Means)
- Anomaly Detection
- Recommender Systems
- Principal Component Analysis (PCA)
